# lmtravel
LogMyTravel é um aplicativo móvel projetado para ser o diário de bordo pessoal e digital do usuário. Seu principal objetivo é permitir que o usuário registre, organize e reviva todas as suas experiências de viagem de forma intuitiva, privada e com ênfase na coleta de dados relevantes (localização, data, despesas, notas e fotos).
